    Planejamento lança novo Portal do Software Público Brasileiro
    Ambiente traz soluções que dispensam pagamentos de licença simples, de fácil utilização e com novas funcionalidades que permitem o desenvolvimento colaborativo de soluções públicas e uma melhor interação entre os usuários. Este é o novo Portal do Software Público Brasileiro, lançado em versão beta nesta terça-feira (9), durante o Seminário de Apoio Tecnológico aos Municípios, em Brasília.
