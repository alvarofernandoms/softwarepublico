Como enviar mensagens para as listas?

Para enviar uma nova mensagem para as listas, basta enviar um e-mail para:

{profile}@listas.softwarepublico.gov.br

Responder uma mensagem existente:

Basta responder o e-mail que foi recebido ou acesse o Portal SPB e faça o login com seu usuário e senha.
